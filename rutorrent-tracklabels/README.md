*Track Labels* plugin adds a set of labels on the category panel. These labels are created automatically on the basis of torrents' trackers.

- - - -

You can override **DOCROOT**, **PHPUSER**, and **PHPGROUP** at build time.
They default to:
* DOCROOT=/var/www
* PHPUSER=root
* PHPGROUP=apache

**Requires**: [rutorrent](rutorrent)
- - - -

For configuration help, you can visit:
https://github.com/Novik/ruTorrent/wiki/PluginTracklabels
